Fast Food or Vegetarian

See how many fast food and vegetarian restaruants are in each states.

Kwon, YoungJu - CS498 Data Visualization

About the Visualization

This web is interactive slideshow based on the data of number of vegetarian and fast food restaurants. Dataset contains each restaurants’ locations, states they located in, menus, name and so on. With population dataset added, the numbers are scaled for better comparison between states.

Overall Structures

onepage-scroll.js template is employed for interactive slideshow. It basically enables to users to go slide by slide with simple scrolling or direction keys.

Scenes

The project consists of slides. Users can navigate each slide in sequence by using scrolling, or up down buttons. Also, on the right side of web, it has small dots for navigating pages. In order to see details of each data of states, I use different types of charts including scatter, choropleth map, and bar charts. I found out scatter or bar chart alone is not enough to give location information about states.

Annotations

Various tooltips, axis, labels and legends are used in all three different scenes. For instance, in scatter plot, tooltips come up when mouse is on the circles, describing its population and number of each types of restaurants.

Parameters

On the right side of web, it has small dots for navigating pages. In scatter plot, legend boxes are parameters for filtering each region of states. By clicking each states in the choropleth map, bar plot shows number of each types restaurants of selected states.

Triggers

Zoom and buttons are employed for updating new parameters in scenes. By double clicking and dragging, you can zoom into details of scatter plot data, with coordinates parameters are updated. And ‘reset zoom’ button will trigger to return to the original coordinates and sizes. ‘Vegetarian’, ‘Fast-Food’ buttons are triggers that change bar plot with its maximum and minimum numbers.
