Fast Food or Vegetarian

See how many fast food and vegetarian restaruants are in each states.

Kwon, YoungJu - CS498 Data Visualization

About the Visualization

This web is interactive slideshow based on the data of number of vegetarian and fast food restaurants. Dataset contains each restaurants’ locations, states they located in, menus, name and so on. With population dataset added, the numbers are scaled for better comparison between states.

Overall Structures

onepage-scroll.js template is employed for interactive slideshow. It basically enables to users to go slide by slide with simple scrolling or direction keys.

Scenes

The project consists of slides. Users can navigate each slide in sequence by using scrolling, or up down buttons. In order to explore both overview and details of each data of states, various types of charts for scenes including scatter, choropleth map, and bar charts are used. Scatter or bar chart alone is not enough to give location information about states.
Scatter plot shows general information about the number of each restaurants with and without scaled by population. Axes are little larger than range of data for better view. Axes also change when scaled. The size of mark is proportional to it population.
Choropleth map and bar charts are grouped together in fourth slide. Map visualization give better intuition about which state has how many restaurants. And with bar plots, it is easier to compare with maximum and minimum numbers of restaurants.

Annotations

Various tooltips, axis, labels and legends are used in all three different scenes. For instance, in scatter plot, tooltips come up when mouse is on the circles, describing its population and number of each types of restaurants.

Parameters

On the right side of web, it has small dots for navigating pages.
In scatter plot, legend boxes are parameters for filtering circles based on regions of states.
By clicking each states in the choropleth map, bar plot shows number of each types restaurants of selected states in bar plot.

Triggers

Zoom functions and buttons are employed for updating new parameters in scenes. By double clicking and dragging, coordinates parameters are updated. And ‘reset zoom’ button will trigger to return to the original coordinates and sizes, while ‘scale’ button moves each circles position to scaled coordinates.
‘Vegetarian’, ‘Fast-Food’ buttons under choropleth map are triggers that change bar plot with its maximum and minimum numbers based on the type of restaurants.
